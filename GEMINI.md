# GEMINI

## Skillset

### Frontend (Flutter)
- **Framework**: Flutter (Single codebase for iOS/Android)
- **Language**: Dart
- **State Management**: Riverpod
- **Routing**: GoRouter
- **Animation**: flutter_animate, Rive (for character animations)

### Backend (BaaS - Supabase)
- **Platform**: Supabase
- **Database**: PostgreSQL
- **Serverless**: Supabase Edge Functions (TypeScript/Deno)
- **Auth**: Supabase Auth (Kakao, Apple, Google, Anonymous)
- **Storage**: Supabase Storage

### Mobile Services
- **Push Notifications**: Firebase Cloud Messaging (FCM)
- **Ads**: Google Mobile Ads (AdMob)
- **Payments**: In-App Purchase (flutter_inapp_purchase)

### Analytics & Monitoring
- **Analytics**: Firebase Analytics, Amplitude
- **A/B Testing**: Firebase Remote Config
- **Error Tracking**: Sentry

### Domain Knowledge
- **Saju (Four Pillars of Destiny)**: Understanding of heavenly stems, earthly branches, five elements, and fortune calculation logic.
- **Tarot**: Understanding of Tarot card meanings (upright/reversed) and spread interpretation.
