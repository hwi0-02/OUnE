2. 발견된 치명적 데이터 오류 (수정 필수)
lib/core/data/monthly_solar_terms_data.dart 파일의 2025년 데이터에 오류가 있습니다.

현상:

2024년 데이터의 12월(소한)은 [1, 6...] (1월 6일)로 2024년 1월을 가리킵니다.

그러나 2025년 데이터의 12월(소한)은 [2026, 1, 5...] (2026년 1월)로 되어 있어, 2025년 1월 데이터가 누락되었습니다.

이로 인해 2024년 12월생이나 2025년 1월생의 대운 계산 시, 다음 절기를 찾지 못해 **약 365일의 오차(대운수 120세)**가 발생하게 됩니다.

수정 코드 (monthly_solar_terms_data.dart): 2025년 데이터의 12번 키 값을 **2025년 1월 5일(소한)**로 수정해야 데이터의 일관성이 유지됩니다.

Dart

    2025: {
      1:  [2, 3, 22, 7],
      // ... (중간 생략) ...
      11: [12, 7, 5, 5],
      // [오류] 12: [2026, 1, 5, 23, 1],  <-- 2026년 소한 (잘못된 값)
      // [수정] 2025년의 소한(1월) 데이터가 들어가야 합니다.
      12: [1, 5, 17, 55],  // 2025년 1월 5일 17시 55분 (소한)
    },
    // 필요하다면 2026년 데이터를 추가해야 합니다.
    2026: {
      12: [1, 5, 23, 1], // 2026년 1월 5일 (소한)
      // ...
    }
